Для установки агента обновления u2u1 под Windows необходимо:


1) Если пинпад подключен через СОМ- или USB-порт, то
  1.1) Номер СОМ-порта, к которому подключен пинпад, указать в файле pinpad.ini как обычно. 

2) Убедиться, что адрес 185.157.96.41:80 доступен с кассы.

3) Выполнить сверку итогов для проверки работоспособности через cmd Loadparm 7. При этом в каталоге должен появиться подкаталог Update. Сверка итогов должна пройти успешно, без ощутимых задержек.
